PROFILE

Radja Boy

Nama saya Radja Boy. Saya lahir di Jakarta, pada tanggal 16 Agustus 2001. Saya adalah anak kedua dari dua bersaudara kembar. Saya sekarang berkuliah di Universitas Advent Indonesia (UNAI), mengambil jurusan Sistem Informasi. Di kala pandemi virus COVID-19 yang sedang berlangsung ini, saya bertempat tinggal di rumah, yaitu di Cibubur, Depok. Hobby saya adalah berolahraga dan juga bermain musik. Di olahraga, saya senang bermain bulutangkis, dan kadang-kadang saya juga bermain basket. Untuk musik, saya bisa bermain piano, gitar, saxophone. Tapi untuk musik, itu menjadi hobi minoritas saya. Saya beragama Kristen Protestan, dengan lebih tepatnya adalah Kristen Advent. Saya berjemaat di Pasar Minggu. Di UNAI, saya adalah angkatan 2019, dimana saya merupakan tamatan tahun 2019 dari sekolah menengah atas. Awalnya saya berminat untuk mengambil salah satu universitas negeri di Malang, tetapi oleh karena satu dan dua hal, sehingga saya akhirnya berkuliah di universitas saya sekarang, yaitu Universitas Advent Indonesia.
